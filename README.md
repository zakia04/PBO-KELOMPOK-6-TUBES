# Rental Gradle

A rental management application built using Java and Gradle.  
This project demonstrates a modular application structure with Gradle build automation.

## Features

- Customer management
- CRUD operations
- Gradle-based build system
- Clean project structure

## Tech Stack

- Java
- Gradle
- MySQL / PostgreSQL (update based on your DB)

## How to Build
```console
gradlew build
```

## How to Run
```console
gradlew run --console plain
```
